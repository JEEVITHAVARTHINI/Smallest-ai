## 🌐 Hosting the Docs

You can host this documentation using **GitHub Pages**:

1. Push this folder to a GitHub repository.
2. Go to the repository’s **Settings > Pages**.
3. Under **Source**, choose the `main` branch and set the folder to `/smallest-ai`.
4. Save — GitHub will provide a public URL.

## 📌 How to View Locally

Simply open the `index.html` file in your browser to view the documentation using Redoc.
